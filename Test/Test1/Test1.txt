Ques 1. What is HTML?
Answer 1. HTML stands for Hyper Text Markup Language. It is the standard Markup Language used to create a web page. 
HTML is a combination of hyper text and Markup where hyper text defines the link between web pages and markup language is used to define the document within the tag to define structure of web page.
It contains HTML tags and attributes to describe format and structure.

Ques 2.  What are the features of HTML?
Answer 2 The features of HTML are as follows:
1. It is easy to learn and understand.
2. It is platform independent.
3. Images,audio,video will be added to a web page using HTML.
4. It is case insensitive.

Ques 3. What is MIME?
Answer 3. MIME stands for Multipurpose Internet Mail Extensions. It takes care of transferring different types of data such as audio, video, image through e-mail services. It is used in web as well as on emails.
