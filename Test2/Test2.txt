Q1.What is CSS? What are the different types of selectors in CSS?
Answer1. It stands for Cascading Style Sheet. It is simple design language intented to simplify the process of making web pages presentable.
It is easy to manage and saves a lot of time.
Selectors in CSS are used to find the HTML elements we want to style.
There are many types of selectors and contain any tag from the html like <h1>,<p>,<table> etc.

Syntax of selector -
selector{
    property:value;
}
